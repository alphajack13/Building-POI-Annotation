# Building-POI-Annotation
Interaction for Building-POI Category Annotation

These are the source codes for the **Geometry-aware Spatial-Textual Learning framework for building-aware POI category prediction** model and its corresponding data.

- Data and pretrained *Prompt-guided Qwen3 LLM Textual Encoder*
   https://pan.baidu.com/s/15Uy6mwtI1AzA6eOB5Ct_uw?pwd=kpcd 
  
- Code

  1. src/models/BuildingPA.py is the code of the main structure of the **Geometry-aware Spatial-Textual Learning framework for building-aware POI category prediction** model.
  
  2. src/main.py is executed to train the model.
  
  3. POI name encoding model see: 
  
     https://huggingface.co/Qwen/Qwen3-Embedding-4B
  
  4. Base semantic model see: 
  
     https://huggingface.co/Qwen/Qwen3-1.7B
